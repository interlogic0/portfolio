# InterLogic Portfolio

Futuristic portfolio site showcasing SaaS, newsletter, AI, and mobile apps built by [InterLogic](https://github.com/interlogic0).

## Live site

Deploy on [Vercel](https://vercel.com) and set your production URL in the badge below:

[![Deployed on Vercel](https://img.shields.io/badge/Deployed-Vercel-black?style=flat&logo=vercel)](https://vercel.com)

> After your first deploy, replace the link above with your live URL (for example `https://portfolio-interlogic0.vercel.app`).

## Stack

- **Next.js 15** (App Router, static export-friendly build)
- **TypeScript**
- **Tailwind CSS v4**
- **Framer Motion**

## Local development

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).

## Deploy to Vercel

1. Import this repository at [vercel.com/new](https://vercel.com/new).
2. Framework preset: **Next.js** (auto-detected).
3. No environment variables are required for this site.
4. Deploy.

Build command: `npm run build`  
Output: Next.js default (`.next`)

## Security

This project is a **public marketing portfolio**. It does not use API keys, databases, or server secrets. Never commit `.env` files or credentials — see [SECURITY.md](./SECURITY.md).
