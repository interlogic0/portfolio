# Security Policy

This repository is a **public portfolio website**. It intentionally contains:

- Public project names, descriptions, and live website URLs
- Links to public GitHub profile and deployed apps

It must **never** contain:

- API keys, tokens, or passwords
- Private `.env` or `.env.local` files
- Internal infrastructure URLs, database credentials, or SSH keys

If you believe sensitive data was committed by mistake, contact the repository owner and rotate any exposed credentials immediately.

## Reporting

Open a private GitHub security advisory on this repository if you discover a security issue.
